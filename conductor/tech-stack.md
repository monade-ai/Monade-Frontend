# Tech Stack

## Frontend
- **Framework:** Next.js 15 (App Router)
- **Library:** React 19
- **Styling:** Tailwind CSS 4, PostCSS
- **Animations:** Framer Motion
- **3D Graphics:** Three.js
- **Icons:** Lucide React

## Development Tools
- **Language:** TypeScript
- **Linting:** ESLint
- **Package Manager:** npm
